Bob O’Hara

Introduction

This is an implementation of the diversity statistics, to demonstrate their calculation without any trait or environmental variables. We will use data where we can estimate both occurrence and abundance (conditional on occurrence).

The Community Model

For species \(i\) on site \(j\) we denote occurrence as \(z_{ij} \in \{0,1\}\) (where 1 is presence), and abundance as \(\lambda_{ij} > 0\). We will link both of these to the general theory, where we assume the occurrence/abundance/whatever is \(k_ij\) for species \(j\) on site \(i\).

Fo occurrence we assume presence is a Bernoulli trial, then \(z_{ij} \sim \text{Bern}(p_{ij})\). with probability \(\text{logit}(\eta_{ij}) = \log\frac{p_{ij}}{1-p_{ij}}\). This is equivalent to a threshold model, i.e. saying that we have a continuous random variable, \(\eta_{ij}^{(z)}\), and \(z_{ij} = 1\) if \(\eta_{ij}^{(z)}>0\). We will assume that \(\eta_{ij}^{(z)}\) follows a logistic distribution, which is equivalent to using a logit link in a GLM. Then

\[ \eta_{ij}^{(z)} = \mu^{(z)} + \alpha^{(z)}_i + \beta^{(z)}_j + \gamma^{(z)}_j x_i + \delta^{(z)}_{ij}. \]

And \(\delta^{(z)}_{ij}\) follows a standard logistic distribution, which means it has a mean of 0 and variance \(\pi^2/3\). We can calculate statistics on \(\eta_{ij}^{(z)}\): note that \(z_{ij}|\eta_{ij}^{(z)}\) is deterministic: the usual Bernoulli stochasticity is in \(\delta^{(z)}_{ij}\).

For abundance we assume a log-linear model:

\[ \log(\lambda_{ij}|z_{ij}=1) = \eta_{ij}^{(\lambda)} = \mu^{(\lambda)} + \alpha^{(\lambda)}_i + \beta^{(\lambda)}_j + \gamma^{(\lambda)}_j x_i + \delta^{(\lambda)}_{ij} \]

Clearly \(\lambda_{ij}=0\) if \(z_{ij}=0\).

For both models we assume the effects are random effects, i.e. with a variance to be estimated. But to make the presentation easier to follow, we use the posterior estimates of the parameters to calculate variances from.

The models are fitted with JAGS, so it is easier to estimate uncertainty. Detailsa re in the fold-out below.

An Example

NoteObservation Model

The data are counts of each species, and we model both presence/absence and counts. We have multiple replicate observations per site, so we assume the counts, \(n_{kj}\) on replicate \(k\) for species \(j\), follow a zero inflated Poisson lognormal distribution with mean \(\lambda_{ij}\) (i.e. constant across replicates within a site). If the detection probability is constant within a species (but differing between species) this will change the \(\beta^{(\lambda)}_j\) term. We have a Gaussian overdispersion term for excess variation within a replicate that represents sampling variation.

Downloading and formatting the data

Diversity Output

All of our diversity metrics are based on decomposing the distributions into:

  • Occurrence
  • Abundance

And looking at the means and variances of the parameters associated with these.

Within-Site Diversity

Within-Site Means: \(\beta\)-mean diversity

The overall means are probably rather boring, but we can look at the site means. For abundance we only average over the species that are present: for those we do not observe, we estimate presence.

Sites <- 1:DataToJAGS$NSite
Species <- 1:DataToJAGS$NSp

# Function to Calculate mean occupancy or abundance for each site
CalcSiteMean <- function(site, mc, Occ=TRUE) {
# Extract relevant statistics from MCMC draws
  mu <- mc[,grep(paste0("^logmu\\[", site, ","), colnames(mc))]
  inc <- mc[,grep(paste0("^Present\\[", site, ","), colnames(mc))]
  p <- mc[,grep(paste0("^p\\[", site, ","), colnames(mc))]
  
  if(Occ) {
    res <- apply(log(1/(1/p - 1)), 1, mean)
  } else {
    res <- sapply(1:nrow(mc), function(i, mu, pres) {
      mean(mu[1,pres[i,]==1]) # mean for when presence=1
    }, mu=mu, pres=inc)
  }
  res
  
}

# Function to calculate species richness
CalcSR <- function(site, mc) {
  inc <- mc[,grep(paste0("^Present\\[", site, ","), colnames(mc))]
  res <- apply(inc, 1, sum)
  res
}

SpRichness <- coda::as.mcmc(sapply(Sites, CalcSR, mc=DivPars))

PlotPars <- function(Summ, At="Orig",...) {
  if(length(At)==length(Summ$PMode)) {
    At.y <- At
  } else {
    At.y <- switch(At, 
                   Orig = 1:length(Summ$PMode), 
                   rank = rank(Summ$PMode, ties="random")
    )
  }
  plot(Summ$PMode, At.y, xlim=range(Summ$HPD), ...)
  segments(Summ$HPD[,"lower"], At.y, Summ$HPD[,"upper"], At.y)
  
}

Unevenness: α-var-diversity

Our theory says that unevenness is \(\text{Var}_j(k_{i,j})\), i.e. the variance in the species’ effects on each site. For abundance this is related to measures such as Shannon’s and Simpson’s diversity: the more evenly distributed species’ abundances are, the higher the abundance (an uneven distribution would mean one or a few species with high abundances, i.e. dominating the community). Our variance is a measure of unevenness: when it is zero, all species have the same abundance, so there is maximal diversity.

In practice we calculate our abundance unevenness only using the species estimated to be present on the site.

Sites <- 1:DataToJAGS$NSite
Species <- 1:DataToJAGS$NSp

CalcUnevenness <- function(site, mc, Occ=TRUE) {
  mu <- mc[,grep(paste0("^logmu\\[", site, ","), colnames(mc))]
  inc <- mc[,grep(paste0("^Present\\[", site, ","), colnames(mc))]
  p <- mc[,grep(paste0("^p\\[", site, ","), colnames(mc))]
  
  if(Occ) {
    res <- apply(log(1/(1/p - 1)), 1, var)
  } else {
    res <- sapply(1:nrow(mc), function(i, mu, pres) {
      var(mu[1,pres[i,]==1])
    }, mu=mu, pres=inc)
  }
  res
  
}

Specificity: β-var-diversity

In practice we calculate our abundance unevenness only using the species estimated to be present on the site.

Sites <- 1:DataToJAGS$NSite
Species <- 1:DataToJAGS$NSp

CalcSpecificity <- function(sp, mc, Occ=TRUE) {
  mu <- mc[,grep(paste0("^logmu\\[.*,", sp, "]"), colnames(mc))]
  inc <- mc[,grep(paste0("^Present\\[.*,", sp, "]"), colnames(mc))]
  p <- mc[,grep(paste0("^p\\[.*,", sp, "]"), colnames(mc))]
  
  if(Occ) {
    res <- apply(log(p/(1-p)), 1, var)
  } else {
    res <- sapply(1:nrow(mc), function(i, mu, inc) {
      mus <- mu[i,inc[i,]==1]
      ifelse(length(mus)>1, var(mus), 0)
    }, mu=mu, inc=inc)
  }
  res
  
}

SpecificityOcc <- sapply(Species[1], CalcSpecificity, mc=DivPars)
SpecificityAbund <- coda::as.mcmc(sapply(Species, CalcSpecificity, Occ=FALSE, mc=DivPars))
